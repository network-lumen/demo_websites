How to go live

1. Prepare the live

Add a title, description, optional cover image, optional offline screen, and choose the audio mode.

2. Grant capture access

The browser asks for the window or screen to share. Audio can come from the shared source, microphone, both, or neither.

3. Create or reuse a stable link

Lumen publishes live resolver records to IPNS. Viewers receive a stable lumen://ipns/... link instead of a changing CID.

4. Start after the countdown

The countdown begins only after permissions and stable link setup are done. Cancelling before it ends prevents chunks from being sent.

5. Share the link

Once live, the Studio shows the share URL and copies it. Viewers opening it automatically enter the player.

6. Offline behavior

If chunks stop arriving for a while, the player switches to OFFLINE and shows the custom offline screen when one was provided.
